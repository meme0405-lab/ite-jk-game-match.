# JK Game Match - Versão Estática

Este repositório contém a versão estática do projeto **JK Game Match**, adaptada para funcionar no **GitHub Pages**.

## Sobre o Projeto
O JK Game Match é uma plataforma de matchmaking para jogadores encontrarem parceiros de jogos com base em interesses, plataforma e nível de habilidade.

## Versão Estática (GitHub Pages)
Como o GitHub Pages não suporta processamento no lado do servidor (PHP) ou bancos de dados (MySQL), esta versão foi adaptada:
- **Login:** Simulado via JavaScript (qualquer email/senha funciona).
- **Dados:** Os dados de jogos e jogadores são carregados de arquivos JavaScript estáticos (`js/dados.js`).
- **Persistência:** O estado do usuário é mantido temporariamente no `localStorage` do navegador.
